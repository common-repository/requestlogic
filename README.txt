== Description ==

The requestlogic plugin is meant to be an extension for [RequestLogic](https://www.requestlogic.com).

This plugin is for a 3rd party requestlogic.com site and only shortcodes generated by this page can be used.
You must register a user on requestlogic.com to be able to create a subscribe form and generate a shortcode. These shortcodes can be used on your WordPress page.
Your form will display on your page, and if someone subscribes, you will see the subscriber instantly on requestlogic.com.

Page site: [requestlogic.com](https://www.requestlogic.com)
Privacy Policy: https://www.requestlogic.com/privacy-policy
Terms of Service: https://www.requestlogic.com/terms-of-service

In order to be able to use our plugin, you must do these steps:

- visit requestlogic.com
- create an account
- create a subscribe form
- copy the shortcode and insert it into your page.

== Installation ==

1. Install the plugin
2. Activate the plugin
3. Insert it into your page.
